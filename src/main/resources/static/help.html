Gestionnaire de STB (Spécification Technique des Besoins)

Documentation de l'API

METHOD ROUTE DESCRIPTION FORMAT ATTENDU FORMAT DE RETOUR
GET /stb23/resume/xml Affiche la liste des spécifications stockées Pas de format attendu Format XML
GET /stb23/resume Affiche la liste des spécifications stockées Pas de format attendu Format HTML
GET /stb23/xml/{id} Affiche le contenu complet de la spécification dont l'identifiant est {id} ID d'une spécifications Format XML
GET /stb23/html/{id} Affiche le contenu complet de la spécification dont l'identifiant est {id} ID d'une spécifications Format HTML
POST stb23/insert Permet d'insérer une spécification Format XML, décrivant la spécification conforme au schéma xsd Format XML, avec le status et l'id de la nouvelle spécification
DELETE stb23/delete/{id} Permet de supprimer une spécification ID d'une spécification Format XML, avec le status et l'id de la spécification supprimé
