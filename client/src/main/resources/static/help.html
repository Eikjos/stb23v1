Gestionnaire de STB (Spécification Technique des Besoins)

Documentation du client

METHOD ROUTE DESCRIPTION FORMAT ATTENDU FORMAT DE RETOUR
GET /stb23 Affiche la liste des spécifications stockées Pas de format attendu Format HTML
GET /stb23/{id} Affiche le contenu complet de la spécification dont l'identifiant est {id} ID d'une spécifications Format HTML
GET stb23/create Page permettant d'insérer une spécification depuis un formulaire ou depuis un fichier Ficher XML ou XML HTML
